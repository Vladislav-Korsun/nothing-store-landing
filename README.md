# Nothing Store Landing
A modern responsive landing page for a tech e-commerce store inspired by the Nothing brand.
The project focuses on clean UI, responsive layout, and structured styling using BEM methodology.

## Live Preview
[View Live Demo](https://Vladislav-Korsun.github.io/nothing-store-landing/)

## Design Reference
This project is based on a Mate Academy landing page design concept.

## 🛠 Technologies Used
- HTML5
- SCSS (Sass)
- JavaScript (ES6)
- BEM (Block Element Modifier)
- Gulp
- Git & GitHub
- GitHub Pages (deployment)

## Getting Started
Follow these steps to run the project locally:

### 1. Clone the repository
git clone https://github.com/Vladislav-Korsun/nothing-store-landing.git
### 2. Navigate to the project folder
cd nothing-store-landing
### 3. Install dependencies
npm install
### 4. Run the project
npm start

## Features
- Fully responsive layout (mobile / tablet / desktop)
- Clean and modern UI design
- Product cards with descriptions and pricing
- Category browsing section
- Navigation menu (mobile + desktop)
- Contact form UI
- Smooth hover animations
- Well-structured SCSS with BEM methodology
